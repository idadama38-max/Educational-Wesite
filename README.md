LEARNHUB Website
Overview

LEARNHUB is an online educational platform designed to help students learn programming, design, and technology skills through interactive courses, video lessons, and quizzes.

:Features
User Registration and Login
Course Catalog
Video Lessons
Progress Tracking
Interactive Quizzes
Mobile-Friendly Design
Certificate Generation

:Technologies Used
HTML
CSS

:Installation
Download or clone the project files.
Open the project folder.
Run index.html in your web browser.

:How to Use
Create an account or log in.
Browse available courses.
Select a course and start learning.
Watch lessons and complete quizzes.
Track your learning progress.

:Courses Available
Python Programming
JavaScript Fundamentals
UI/UX Design
Web Development Basics
